Frequency / Velocity / Line Name

Freq. Range GHz: Frequency range in GHz. Note that for some projects the observers may define the redshifted frequency.

Vel. Range m/s: Velocity range of the observed sources. Note that for some projects the velocity are set to 0. by the observers and the frequency is computed with the redshift.

Line Name: Name of the line observed according to the observing setup. Here is a molecule line database where most of the common line names and frequencies are indicated.
