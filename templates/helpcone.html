Source Name / Position

Source Name: The object name. Note that object names are defined by the observers. When searching on this field you can either search on the "IRAM Object Name" or use the SIMBAD/NED name resolver.

RA, Dec (J2000): The right ascension an declination of the target. If a specific coordinate is entered then the Size Box entry must be completed to specify the search range. For TAPAS queries use hh:mm:ss.ss format: e.g. 12:34:56.7. Note that right ascension coordinates must be given in hours but declination in sexagesimal degrees.

Size: A search for observations near a specified RA and Dec requires a size for the Size box in degrees.
